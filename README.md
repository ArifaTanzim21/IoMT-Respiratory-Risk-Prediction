# IoMT-Respiratory-Risk-Prediction
IoMT-based real-time respiratory risk prediction using ESP32, biomedical sensors, Random Forest, Android, and Firebase.

## 📄 Published Paper

**IoMT-Based AI-Driven Respiratory Risk Prediction App with Firebase Integration for Low-Resource Settings**

This repository contains the implementation and supporting code for our research work on a low-cost IoMT-based respiratory health monitoring system using ESP32, MAX30102, LM35, Random Forest, Android, and Firebase.

[Read the Full Paper on IEEE Xplore](https://ieeexplore.ieee.org/abstract/document/11377379)

### Key Features

* Real-time SpO2, heart rate, and temperature monitoring
* ESP32-based biomedical sensing
* Random Forest-based respiratory risk prediction
* TensorFlow Lite on-device inference
* Android mobile application
* Firebase real-time data storage
* Automated email alerts for abnormal conditions
